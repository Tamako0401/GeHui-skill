# GeHui Classroom Skill

An evidence-aware Chinese Culture Overview classroom assistant. It supports source-backed retrieval, ASR-correction proposals, course-study assistance, and clearly labelled classroom-style rewrites. It is not an identity simulation of the teacher.

## Repository contents

- `ge-hui-classroom/`: the installable Skill package, including its instructions, retrieval script, and methodology references.
- `subtitles-raw/` and `hot-words-database/`: local working corpus. These paths are deliberately ignored and are not part of this public repository.

## Data boundary

Do not publish raw classroom recordings, transcripts, student information, or derived datasets unless the necessary permissions and rights review are in place. Any source-backed answer must identify the underlying file and timestamp; a missing match means only that the current local corpus has no match.

## Current status

This repository is an initial, private-source scaffold. Before public release, add a license for original repository material, a source-rights statement, a small evaluated example set, and a contributor guide for corpus annotations.
